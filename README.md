# Лабиринт сокровищ

**«Лабиринт сокровищ»** — это текстовая приключенческая игра, в которой игрок
исследует запутанные коридоры, собирает предметы, решает загадки,
избегает ловушек и пытается выбраться живым.

В проекте реализованы:
- система комнат и переходов между ними;
- управление состоянием игрока;
- инвентарь и предметы;
- загадки с альтернативными ответами;
- ловушки и условия поражения;
- победные условия.

Проект создан в рамках учебного задания.

---

## Установка

Убедитесь, что у вас установлен **Python 3.12+** и **Poetry**.

Затем выполните:
make install

Эта команда установит зависимости и создаст виртуальное окружение.

---

## Запуск игры

make project

Игра запустится в консоли.

## Структура проекта

labyrinth_game/
│── __init__.py          # Инициализация пакета
│── main.py              # Точка входа, запуск игры
│── constants.py         # Константы, описание комнат, предметов и загадок
│── player_actions.py    # Действия игрока: перемещение, использование предметов
│── utils.py             # Вспомогательные функции (описания, случайные события)
│── __pycache__/         # Служебные файлы Python
│
├── pyproject.toml       # Конфигурация проекта
├── README.md            # Описание проекта
└── Makefile             # Команды установки, запуска, сборки
